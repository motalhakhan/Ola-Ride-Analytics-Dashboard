# Ola Ride Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes Ola ride booking data to uncover trends in revenue, cancellations, vehicle performance, and customer ratings.

## Tools Used
- Power BI
- DAX
- Excel/CSV

## Dashboard Pages

### 1. Overall Dashboard
Provides a summary of bookings, revenue, ride status, and key KPIs.

### 2. Vehicle Type Analysis
Compares booking trends and performance across different vehicle categories.

### 3. Revenue Analysis
Tracks revenue generation and identifies top-performing segments.

### 4. Cancellation Analysis
Analyzes customer and driver cancellations to identify operational issues.

### 5. Rating Analysis
Examines customer ratings and service quality trends.

## Key Insights
- Identified highest revenue-generating vehicle categories.
- Analyzed cancellation patterns affecting business performance.
- Evaluated customer satisfaction using rating metrics.
